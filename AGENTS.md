# Project Rules

- Bu projede degisiklik yapildiginda gerekli kontroller calistirilir.
- Kontrollerden sonra uygun degisiklikler otomatik stage edilir ve aciklamali commit olusturulur.
- Commit mesajlari kisa, Turkce ve mumkunse conventional commit biciminde yazilir.
- Final cevapta yapilan kontroller, commit hash'i ve kalan git durumu ozetlenir.
- Kullanici ayrica istemedikce build/IDE ciktilari ve kullaniciya ozel dosyalar commit edilmez.
